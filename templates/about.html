Posture Corrector was created by Sophie Ma for CAS Comp Sci. The goal was to use Teachable Machine's pose model to create a machine learning model that recognises when the user has bad posture, and plays an alarm in order to alert them. Shoutout to the creators of Teachable Machine, Paul Way, and to the YouTube account "Sound effects" for the sound effects!
